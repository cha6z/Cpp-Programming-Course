## Key C++ Concepts

### Program Structure

Every standard C++ program begins execution from the `main()` function.

A simple C++ program usually contains:

- Header includes
- A `main()` function
- Statements inside the function body
- A return value indicating program execution status

---

### Compilation

C++ source code must be compiled before it can be executed.

The development process usually involves:

```text
Source Code
    ↓
Preprocessing
    ↓
Compilation
    ↓
Linking
    ↓
Executable Program
````

---

### Header Files

Header files provide declarations for functions, classes, constants, and standard library objects.

Common examples include:

| Header        | Purpose                  |
| ------------- | ------------------------ |
| `<iostream>`  | Console input and output |
| `<string>`    | Text strings             |
| `<vector>`    | Dynamic arrays           |
| `<cmath>`     | Mathematical functions   |
| `<fstream>`   | File input and output    |
| `<algorithm>` | Standard algorithms      |

---

### Namespaces

C++ uses namespaces to organise identifiers and avoid naming conflicts.

The standard library is placed inside the `std` namespace.

Example:

```cpp
std::cout << "Learning C++" << std::endl;
```

---

### Functions

Functions are reusable blocks of code that perform specific tasks.

They improve:

* Readability
* Maintainability
* Reusability
* Program structure
* Debugging

---

### Pointers and References

C++ provides direct access to memory through pointers and references.

This is one of the most powerful parts of the language, but also one of the easiest places to introduce bugs.

Pointers and references are important for understanding:

* Memory addresses
* Function parameters
* Dynamic memory allocation
* Object manipulation
* Embedded and robotics programming

---

### Object-Oriented Programming

C++ supports object-oriented programming through classes and objects.

Core concepts include:

* Classes
* Objects
* Constructors
* Destructors
* Encapsulation
* Inheritance
* Polymorphism
